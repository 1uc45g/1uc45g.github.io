Projet : Développement d'une application de chat en ligne.

Objectif du projet : développement d'une application de chat en ligne multi-utilisateurs en java.

À la suite du rendu de ce projet, si le temps nous le permet, nous comptons l'améliorer sur le plan des fonctionnalités ainsi que sur le plan esthétique.
Projet effectué en duo, en 2024
  • Création d'une interface utilisateur permettant l'authentification et l'inscription des utilisateurs, puis la récupération et l'affichage des messages en temps réel via un serveur multithread, le tout programmé en java.
  • Documentation du projet en rédigeant des commentaires javadoc adaptés au code et en présentant le projet dans sa globalité sur GIT via un fichier README.
    • Consulter l'espace Github
  • Préparation d'une base de données mySQL adaptée au projet.
  • Ce projet m’a permis d’apprendre à travailler les compétences d'installation, d'adaptation et de développement d'outils logiciels permettant l’échange d’informations au sein de l’entreprise; ainsi que la mise en œuvre de différents protocoles de communication, différents moyens d’authentification et de sauvegarde des données sécurisées; mais aussi à faire face aux imprévus.

Project: Development of an online chat application.

Project objective: development of a multi-user online chat application in java.

Following the rendering of this project, if time permits, we intend to improve it in terms of functionality and aesthetics.
Project carried out in duo, in 2024
  • Creation of a user interface allowing authentication and registration of users, then retrieval and display of messages in real time via a multithreaded server, all programmed in java.
  • Preparation of a project-specific mySQL database.
  • Project documentation by writing code-adapted javadoc comments and presenting the entire project on GIT via a README file.
    • See Github space
  • This project allowed me to learn how to work on the skills of installation, adaptation and development of software tools for the exchange of information within the company; as well as the implementation of different communication protocols, different means of authentication and backup of secure data; but also to deal with unexpected events.
