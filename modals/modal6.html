Projet : Mise en place d'un serveur de fichiers

Objectif du projet : Mettre en place et surveiller des sauvegardes automatiques régulières de contrôleurs WiFi.

Projet conduit dans le cadre d'un stage en entreprise.
Projet effectué seul, en 2024
  • Configuration d'un serveur SFTP sous linux en utilisant une machine virtuelle gérée avec HYPER-V.
  • Création de règles de pare-feu, permettant la connectivité.
  • Ajout de sécurité avec l'application "Fail2ban", configurée afin de bloquer des adresses IP en cas de trop nombreuses tentatives infructueuses de connexion, empêchant ainsi les tentatives d'attaque par force brute ou par déni de service.
  • Documentation du projet en réalisant notamment un schéma réseau complet, et en répertoriant les accès.
  • Attribution des permissions afin de respecter le principe de privilèges moindres.

Project: Setting up a file server

Project objective: Setting up and monitoring regular automatic backups of WiFi controllers.

Project conducted as part of an in-company internship
Solo project, in 2024
  • Configuration of a SFTP server using linux on a virtual machine, managed with HYPER-V.
  • Creation of firewall rules, allowing connectivity.
  • Adding security with the "Fail2Ban" application, configured to block IP addresses in case of too many unfructuous connection requests, preventing brute force ou denial of service attack attempts.
  • Documentation of the project by making a complete scheme, and by listing the accesses.
  • Attributing permissions in order to respect the principle of least privileges.
