Projet : Découverte du web hacking et de la sécurisation du web

Objectif du projet : Découverte du web hacking en exploitant les failles sur une machine volontairement vulnérable (DVWA : Damn Vulnerable Web Application) puis en produisant un rapport écrit ainsi qu'oral sur notre exploitation.

Project : Discovering web hacking

Project objective : Discovery of web hacking in exploiting the vulnerabilities on a deliberately vulnerable machine (DVWA: Damn Vulnerable Web Application) then by producing a written and oral report on our exploitation.

Projet : Conception du réseau informatique d'une petite entreprise

Objectif du projet : Conception du réseau local d'une petite entreprise, sur Cisco Packet Tracer, pour répondre à ses besoins en commutation, routage, services réseau de base et sécurité.

Project : Small Business Computer Network Design

Project objective : Design a small business LAN on Cisco Packet Tracer to meet its switching, routing, core network services and security needs.

Projet : Traitement des données de fichiers de log

Objectif du projet : création d'un programme en python, générant un «dashboard» (tableau de bord) qui affiche des informations et des graphiques calculés à partir de données tirées d'un fichier de log.

Project : Log file data processing

Project objective : creation of a program in python, generating a dashboard that displays information and graphs calculated from data from a log file.
